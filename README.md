# xpay-wechatpay [![Build Status](https://travis-ci.org/xxd3vin/xpay-wechatpay.svg?branch=master)](https://travis-ci.org/xxd3vin/xpay-wechatpay)

## Usage

```javascript
var gateway = require('xpay-wechatpay');
gateway.setX();
gateway.setY();
var order = {};
gateway.purchase(order);
```
